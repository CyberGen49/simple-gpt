SimpleGPT

Interact with OpenAI's language models with ease!

Start by setting your OpenAI API key in Settings.

Messages you exchange with the AI here are only sent between your browser and OpenAI. Your message history and settings are stored locally in your browser.

Images you send to the AI will first be uploaded to ImgBB using the API key in Settings. With the provided key, your images may be seen by others. Use your own key to prevent this.

Check out the project's GitHub repo to learn more.
